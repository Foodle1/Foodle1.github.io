AI Pathfinding

This is prototype to demonstrate AI pathfinding using A* and Greedy

❮ ❯

Project Overview

I developed this project at QUT for the unit IGB383: AI for Games. The aim of the project is to incorporate and understand common AI pathfinding methods, A* and Greedy, and when and where they should be used. The NPCs also had finite state machines controlling their logic.

This project was made in Unity and programmed in C#. A* and Greedy were both adapted from common pseudocode to work within the project.
