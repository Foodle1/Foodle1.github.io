Starfighter: Earth's Only Hope

This is a VR action spaceflight game developed during IGB388 Design and Development of Immersive Environments

❮ ❯

Project Overview

During development of Starfighter: Earth's Only Hope I Acted as lead programmer, working with another programmer and a game designer. This game started as my own pitch, then evolving into the game it is after incorporating elements from the rest of the groups pitches and ideas.

This project was made in Unity and programmed in C#. We opted to use the OpenXR framework for interacting with the VR headset within the project. This allowed me to further my Unity/C# skills while learning about development for VR and all the challenges within that field. This also allowed for me to gain experience developing for multiple platforms, Desktop (PCVR) and Android (Quest Standalone).

  • During this project, I undertook the following roles/tasks:
  • Lead Programmer
  • Concept/Pitch Designer
  • Playtesting
